## ¿Qué es CuentasClaras?

CuentasClaras nace para resolver un problema real: muchos pequeños comercios en España siguen gestionando su inventario con libretas o hojas de cálculo. Esto genera errores, roturas de stock y pérdidas económicas evitables.

La app ofrece una solución digital, sencilla y accesible desde el móvil que cualquier comerciante puede usar desde el primer día, sin formación técnica.

---

## Funcionalidades

| Módulo | Descripción |
|---|---|
| **Dashboard** | Resumen del día: ventas, ingresos, alertas y gráfica semanal |
| **Productos** | Catálogo completo con CRUD, búsqueda, filtros por categoría y ajuste de stock |
| **Ventas** | Registro de ventas con carrito, métodos de pago e historial |
| **Alertas** | Notificaciones automáticas de stock bajo y productos agotados |
